    • Email Sending

    Please read this part carefully because it is one of the most vulnerable part of backend side. You must secure this.

    Manage Your Google -> Account -> Data&Security -> Two Factor Authentication -> App Password

    Create a .env type file

    `.env` ```txt EMAIL=yourMailAddress EMAIL_PASSWORD=yourAppPassword `` `setting.py:` ```py EMAIL_BACKEND = 'django.core.mail.backends.smtp.EmailBackend' EMAIL_HOST = 'smtp.gmail.com' EMAIL_USE_TLS = True EMAIL_PORT = 587 EMAIL_HOST_USER = env(EMAIL) #sender's email-id (your email id) EMAIL_HOST_PASSWORD = env(EMAIL_PASSWORD) #password associated with above email-id (app password) ```
